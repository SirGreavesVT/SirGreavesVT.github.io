Drawing Overlay

Copy [this] link and paste it into an OBS browser source set to your stream's resolution. Make sure it's under any overlays or vtuber models, but over any game or monitor input sources.

Escape clears the whole canvas. Right-click toggles the palette. Mousing-out hides the overlay. Scroll-wheel changes brush size. There is no opacity, but there are three draw-modes. (Buttons under the palette.)
