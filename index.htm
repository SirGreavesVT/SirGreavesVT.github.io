Drawing Overlay

If you're here for the drawing overlay I've been working on, read this first:

Copy [this] link and paste it into an OBS browser source set to your stream's resolution. Make sure it's under any overlays or vtuber models you have set up, but over any game or monitor input sources.

That's it! You can also download the file and link it locally in OBS, but copying that link will ensure you receive any updates or bugfixes automatically.

Things to keep in-mind:

To interact with it, you have to right-click the browser source and click "interact".

Escape clears the whole canvas. Right-click shows and hides the palette. Mousing-out hides the overlay. Scroll-wheel changes brush size. There is no opacity, but there are three draw-modes. (Buttons under the palette.)
